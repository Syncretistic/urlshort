FreeCodeCamp API Basejump: URL Shortener Microservice.

User stories:

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link


How to get the shortened URL:

---> https://urlshort-syncretistic.c9users.io/shorten/http(s)://(www.)exampleurl.com

Result:

{
normal: "http(s)://(www.)exampleurl.com",
shortened: "https://urlshort-syncretistic.c9users.io/short/12"
}


Use the shortened link to be redirected to the corresponding webpage.
